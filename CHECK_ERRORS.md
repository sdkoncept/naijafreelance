# Quick Error Check

## On Your Live Site:

1. Open: http://naijafreelance.sdkoncept.com
2. Press F12
3. Go to Console tab
4. Try to place an order
5. Copy any RED error messages

## Common Errors:

- "Invalid API key" → Need LIVE key, not TEST key
- "Test key used in live mode" → Need LIVE key
- "Paystack public key not configured" → Key not in build
- "Failed to load payment gateway" → Script loading issue
